# arknights_extractor

## なにこれ

- アークナイツの立ち絵データを抽出します

## 環境
- Python 3.8.1
- pipでOpenCVを入れてください

## 使い方

1. InputフォルダーにUnityから取れたpngを入れる
1. extractor.pyを実行し、1を入力
1. 再度extractor.pyを実行し、2を入力
1. Pictureフォルダーに透過pngで書き出されてます
